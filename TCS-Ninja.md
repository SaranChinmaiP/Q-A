# 1. Sweet Seventeen
Given a maximum of four digit to the base 17(10 -> A, 11 -> B, 12 -> C, 16 -> G) as input, output its decimal value.

*Code* :
<code>
  Input : 23GF
  </code>
```
num = '23GF'
print(int(num,17))
```
```
Output: 10980
```
